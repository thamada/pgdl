# /NVMP

------------------------------
## 書式

/NVMP *num*;

------------------------------
## 引数

- *num*: バーチャルマルチプルパイプライン数(パイプラインの多重度) 

------------------------------
## 記述例

```
/NVMP 2;
```


------------------------------
## 概要

/NVMPはバーチャルマルチプルパイプライン数を指定します. 
/NVMPが1については バーチャルマルチプルパイプライン化しないということを意味します.
