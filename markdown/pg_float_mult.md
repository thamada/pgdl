# pg_float_mult : 浮動小数点型 乗算</h3>

## 書式

pg_float_mult(*x*, *y*, *z*,   *NFLO*, *NMAN*, *NST*,*RM*);


## 引数

- *x*:  被乗数
- *y*:  乗数
- *z*:  乗算結果
- *NFLO*: 浮動小数点ワードビット幅
- *NMAN*:  仮数部(精度)ビット幅
- *NST*:  パイプラインステージ数
- *RM*:  丸めモード(省略可) 全9種類. 省略時はTOEVEN.


## 記述例

```
pg_float_mult(x, y, z, 18,  8, 3, 0);
pg_float_mult(x, y, z, 26, 16, 3);
pg_float_mult(x, y, z, 33, 23, 3);
```


## 概要

x, yとの乗算結果をzに渡す.
