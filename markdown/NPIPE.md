# /NPIPE

------------------------------
## 書式

/NPIPE *num*;

------------------------------
## 引数
- *num*: プロセッシングエレメント(パイプライン)の数

------------------------------
## 記述例

```
/NPIPE 10;
```

------------------------------
## 概要

/NPIPEはPEの数を指定します.
