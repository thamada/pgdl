# /CONST_LOG

------------------------------
## 書式

/CONST_LOG *vname*, *value*,  *len*, *mlen*, *scale*;


------------------------------
## 引数

- *vname*: データフロー記述部変数名
- *value*: 定数値 (実数で指定します)
- *len*: 浮動小数点ビット長
- *mlen*: 仮数部ビット長
- *scale*: スケーリング係数

------------------------------
## 記述例

```
/CONST_LOG cx,    -1.5,  17, 8, (pow(2.0,32.0));
/CONST_LOG cm, 7.77e-7, 14 , 5, (pow(2.0,64.0));
```

------------------------------
## 概要

/CONST_LOGは定数をデータフロー記述部で使用するための定数指定コマンドで
す.  基本的な使用目的は/VALSETと同じです. /VALSETとの違いは/CONST_LOG
は定数値を実数で指定できることです. 指定した実数定数は対数型ビット列に
変換されます.
<br><br>

また/CONST_LOGはpgpgプリプロセッサ(pgpp)によって/VALSETへ変換されます.
先の記述例はpgpgプリプロセッサによって一度次のような/VALSETへ変換されま
す.
<br><br>

```
/VALSET cx, 0x1a095, 17;
/VALSET cm, 0x1576,  14;
```
