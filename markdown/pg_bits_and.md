# pg_bits_and : ビット列型 AND

------------------------------
## 書式

pg_bits_and(x, y, z, **NBIT**);

------------------------------
## 引数

- x:  入力ビット列
- y:  入力ビット列
- z:  出力ビット列
- **NBIT**: ビット幅

------------------------------
## 記述例

```
pg_bits_and(x, y, z, 16);
pg_bits_and(x, y, z, 32);
pg_bits_and(x, y, z, 64);
```

------------------------------
## 概要
  x と y の論理積をzへ出力
```
  z  = x & y;
```
