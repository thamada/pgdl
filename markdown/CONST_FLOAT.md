# /CONST_FLOAT

------------------------------
## 書式

/CONST_FLOAT *vname*, *value*,  *len*, *mlen*, *rmode*;

------------------------------
## 引数

- *vname*: データフロー記述部変数名
- *value*: 定数値 (実数で指定します)
- *len*: 浮動小数点ビット長
- *mlen*: 仮数部ビット長
- *rmode*: 丸めモード(省略可) FORCE1, TOEVEN, TRUNCが指定可能

------------------------------
## 記述例

```
/CONST_FLOAT cx, -1.5,     26, 16;
/CONST_FLOAT cm, 7.77e-7,  32, 22;
```

------------------------------
## 概要

/CONST_FLOATは定数をデータフロー記述部で使用するための定数指定コマンド
です.  基本的な使用目的は/VALSETと同じです.  /VALSETとの違いは
/CONST_FLOATは定数値を実数で指定できることです.  指定した実数定数は浮動
小数点型ビット列に変換されます.

また/CONST_FLOATはpgpgプリプロセッサ(pgpp)によって/VALSETへ変換されます.
先の記述例はpgpgプリプロセッサによって一度次のような/VALSETへ変換されます.


```
/VALSET cx, 0x3008000,  26;
/VALSET cm, 0x7ae84985, 32;
```
